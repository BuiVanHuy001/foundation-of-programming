# Singleton Design Pattern

### Singleton Design Pattern là gì?

Singleton pattern là một trong những Creation Pattern. Nó đảm bảo rằng một class chỉ có duy nhất một instance (thực thể)
tồn tại trong suốt vòng đời của ứng dụng, và cung cấp một điểm truy cập toàn cục (global access) tới thực thể này.
